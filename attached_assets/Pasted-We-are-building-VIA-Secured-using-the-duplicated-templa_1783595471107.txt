We are building VIA Secured using the duplicated template.

VIA Secured is a tradesperson verification SaaS platform for UK tradespeople.

IMPORTANT:
Do NOT use Stripe.
Do NOT build Stripe checkout, Stripe webhooks, Stripe products, or subscription API logic.

Use the existing payment-link system already built into the template.

Admin must be able to manually add external payment links for:
1. VIA Membership - £20/month
2. Priority Checking - £49 one-off

Payment links may come from Whop, Stripe Payment Links, PayPal, Lemon Squeezy, or any other provider. The app should simply redirect users to the saved payment URL.

If no payment link is configured, show:
“Payment link not configured yet.”

Core concept:
Tradespeople apply to become VIA checked. Admin verifies their business details, insurance, accreditations, address, digital footprint, public records and contact details. Once approved, the business receives a unique VIA number, public profile, digital badge, and member dashboard.

- Trust/security focused UI
- Professional, serious, credible
- Mobile-first design
